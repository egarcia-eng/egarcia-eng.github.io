# Eduardo Garcia — Engineering Portfolio

Personal portfolio site for Eduardo Garcia, mechanical engineering student
(BSME, University of Texas at Tyler). It showcases finite element analysis,
kinematic design, and mechatronics projects, with the source technical papers
embedded for recruiters to read inline.

## Stack
A single static `index.html` — vanilla HTML, CSS, and JavaScript, no build step
and no dependencies. Figures live in `assets/img/`, project papers and the resume
are PDFs in the project root and in `Side projects/`.

## Run locally
Open `index.html` in any modern browser.
